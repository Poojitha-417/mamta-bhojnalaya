# 🍽️ Mamata Bhojnalaya Website

A responsive restaurant website developed using **HTML** and **CSS** as part of my internship project.

## 📖 About

Mamata Bhojnalaya is a simple and elegant restaurant website that showcases the restaurant's details, menu, contact information, and customer feedback section.

## ✨ Features

- 🏠 Home section with attractive banner
- 👨‍🍳 About Us section
- 🍽️ Restaurant Menu
- 📞 Contact section with clickable phone number
- 📍 Google Maps location link
- 💬 Feedback form
- 📱 Responsive design for desktop and mobile
- Smooth scrolling navigation

## 🛠️ Technologies Used

- HTML5
- CSS3
- GitHub Pages
